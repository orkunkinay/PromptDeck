Quick start

Install from source, build the extension, and load the generated package in a Chromium-based browser.

Shell
git clone https://github.com/orkunkinay/promptdeck.git
cd promptdeck
npm install
npm run build
1Open extensions

Navigate to chrome://extensions.

2Load the build

Enable Developer mode and select dist/.

3Use a command

Open a text field and type ;;.

After rebuilding, reload PromptDeck in chrome://extensions and refresh any pages where the content script should run.

Prompt model

A prompt owns a slash command, optional aliases and tags, immutable versions, and intentional variants. Placeholder tokens are preserved during insertion.

Prompt document
---
command: /paper-reading
aliases: [/paper, /read]
tags: [research, summarize]
---
Summarize the following paper for {{audience}}.

Paper:
{{paper_text}}
ConceptBehavior
CommandsUnique slash identifiers such as /paper-reading
VersionsImmutable snapshots; restore old content as the latest version
VariantsNamed alternatives selected with a suffix such as :short
PlaceholdersTokens such as {{paper_text}} tracked as metadata and preserved at insertion

File library overrides

Use PROMPTDECK_LIBRARY, PROMPTDECK_HOME, or the VS Code promptdeck.libraryPath setting. Platform defaults respect XDG_DATA_HOME and %APPDATA%.

Troubleshooting

Common issues usually come from extension reload state, restricted pages, command collisions, or editor-specific insertion behavior.

The helper does not open

Check the configured trigger, refresh the page after reloading the extension, avoid chrome:// pages, and confirm the current host is not disabled.

A prompt copies instead of inserting

Some rich editors reject synthetic edits. PromptDeck falls back to the clipboard and supports direct, copy-only, and ask-every-time insertion modes.

Save reports a command collision

Commands and aliases must be unique across prompts. Choose another command or remove the conflicting alias.

VS Code is editing the wrong library

Check the path shown in the manager. Override the default with promptdeck.libraryPath, PROMPTDECK_LIBRARY, or PROMPTDECK_HOME.

Browser data is tied to the current profile. Export a JSON backup before clearing data, uninstalling PromptDeck, or changing profiles.
